# **App Name**: Jrk Labor Management

## Core Features:

- Labor Management: Add and manage labor names with their basic details.
- Advance Tracking: Record advance payments given to each laborer, including the date and amount.
- Daily Work Log: Document daily work assignments, noting the location and type of work done by each laborer. Provides means to upload timestamped picture to act as a verification tool of presence.
- Automated Reporting: AI powered tool that automatically generates daily attendance reports. Allows for querying and reporting on total advances paid to labor.

## Style Guidelines:

- Primary color: Forest green (#228B22) to evoke a sense of stability and growth.
- Secondary color: Light grey (#D3D3D3) for backgrounds and neutral elements.
- Accent color: Golden yellow (#FFD700) for important actions and highlights.
- Clean and straightforward layout with clear sections for labor details, advances, and work logs.
- Simple and recognizable icons to represent different actions and categories.
- Subtle animations for adding/removing entries to provide visual feedback.
